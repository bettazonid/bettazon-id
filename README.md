## 🎯 Core Features

1. **Ecommerce Ikan Hias** — listing produk, search/filter by species/variant/size, cart & checkout
2. **Live Commerce** — seller live stream, viewer watch + chat real-time + bid selama live
3. **Page Auction** — lelang di halaman produk dengan timer countdown + chat pembeli
4. **Shipping** — dalam negeri via kurir partner (JNE/SiCepat/etc) + internasional via transshipper Jakarta
5. **Wallet & Escrow** — buyer bayar escrow, release ke seller setelah konfirmasi terima
6. **Rating & Review** — review produk + seller setelah transaksi selesai
7. **Points & Loyalty** — earn points dari transaksi, tukarkan reward digital
